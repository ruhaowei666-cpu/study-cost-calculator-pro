# 🚀 快速启动指南

## 第一步：安装依赖

打开终端（PowerShell或CMD），在项目目录下运行：

```bash
pip install -r requirements.txt
```

如果安装速度慢，可以使用国内镜像：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 第二步：运行应用

### 方式1：使用Streamlit界面（推荐）

```bash
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`

### 方式2：运行测试示例（命令行）

```bash
python test_example.py
```

这会运行三个预设示例，展示不同场景的计算结果。

## 第三步：使用软件

1. **填写表单**：在左侧边栏填写你的信息
   - 选择城市和住宿类型
   - 填写打工信息（如果有）
   - 输入初始存款和学费

2. **点击计算**：点击"🚀 开始计算"按钮

3. **查看结果**：
   - 查看关键指标（月收入、月支出、最终余额等）
   - 查看12个月现金流明细表格
   - 查看现金流趋势折线图
   - 下载PDF报告（可选）

## 📊 示例数据测试

软件内置了三个测试场景，运行 `python test_example.py` 可以查看：

1. **示例1**：有打工收入的学生（里斯本，合租，每周15小时）
2. **示例2**：无打工收入的学生（波尔图，宿舍，一次性学费）
3. **示例3**：资金不足的情况（需要父母支持）

## ⚠️ 常见问题

### 1. 端口被占用

如果8501端口被占用，Streamlit会自动使用下一个可用端口（8502, 8503...）

### 2. 中文显示问题

如果PDF中的中文显示异常，可以：
- 使用 `reportlab` 替代 `fpdf2`（需要安装中文字体）
- 或者只使用英文版本的PDF报告

### 3. 依赖安装失败

如果某个包安装失败，可以单独安装：

```bash
pip install streamlit pandas matplotlib plotly fpdf2
```
